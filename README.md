## Installation & Run
<pre>
git clone https://github.com/asmaagamal871/angularProject.git
</pre>

### Database creation
- create database on mongodb
- create .env


```
DATABASE_NAME= db_name
```

<pre>
cd Angular
npm i
ng serve -o
open your browser on http://localhost:4200/
</pre>

<pre>
cd../Nodejs
npm init
npm install
node index.js
open your browser on http://localhost:3000/
</pre>
